# Telegram Bot (aiogram) для общей игровой учётки

Простой бот на Python + aiogram для работы в группе Telegram.

## Функции

1. Отслеживание, кто играет на общем аккаунте:
   - `/play` — начинает сессиию, сохраняет имя пользователя и время начала.
   - `/stop` — завершает сессию, выводит длительность.
   - `/who` — показывает, кто сейчас играет и сколько времени прошло.

2. Напоминания:
   - `/remind <минуты> <текст>`
   - пример: `/remind 30 фарм`
   - бот через указанное число минут отправит: `Напоминание: фарм`

## Установка

1. Убедитесь, что установлен Python 3.10+ (в вашем окружении уже Python 3.14.3).
2. Установите зависимости:
   ```bash
   pip install aiogram
   ```

## Настройка токена

Вариант 1 (рекомендуется): переменная окружения

- Windows PowerShell:
  ```powershell
  $env:TELEGRAM_BOT_TOKEN = "ВАШ_ТОКЕН"
  python "d:\Dev\TG bot\bot.py"
  ```

- Linux/macOS:
  ```bash
  export TELEGRAM_BOT_TOKEN="ВАШ_ТОКЕН"
  python "d:\Dev\TG bot\bot.py"
  ```

Вариант 2: вставьте токен прямо в `bot.py` (не рекомендуется для публичных репозиториев):

```python
TOKEN = 'ВАШ_ТОКЕН'
```

## Запуск

```bash
python "d:\Dev\TG bot\bot.py"
```

## Примечание для группы

- Добавьте бота в группу.
- Дайте права писать сообщения.
- При необходимости разрешите доступ к сообщениям (для приватных групп и новых Telegram API).

## Как работает память

- Статус текущего игрока хранится в переменной `current_player` в оперативной памяти.
- Проверка свободности происходит в `/play`.
- Для напоминаний используются `asyncio.create_task` + `asyncio.sleep`.
- После перезапуска бота вся история теряется (нет БД).
